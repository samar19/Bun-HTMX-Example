# bun-example-project
## tools
Bun 
HTMX 
Hyperscript 
Pico CSS for the Styling 
Elysia 
Hono 


To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.tsx
```

This project was created using `bun init` in bun v0.8.1. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
